# 🧠 Malayalam Multimodal AI System

A multimodal AI system for Malayalam:

* 📝 Text Classification
* 🖼️ Image OCR Classification
* 🎤 Speech Recognition Classification

## 🚀 Features

* OCR with Tesseract
* Speech recognition using Google API
* Robust NLP using character n-grams

## ▶️ Run

pip install -r requirements.txt
streamlit run app.py

## ⚠️ Note

Malayalam OCR may have limitations due to script complexity.
